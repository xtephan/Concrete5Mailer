Concrete5 Mailer Package
=========

Adds cool mailing functionality to Concrete5.

Version
----

1.0

Requirements
-----------

* Concrete5 5.6.2 to Concrete5 5.7

Installation
--------------

Download in Packages folder and install from dashboard.
